# Código ERAD-RS

Código desenvolvido para uso no minicurso apresentado durante o ERAD-RS 2022.

Para compilar, você precisa do pacote [PMDK](https://github.com/pmem/pmdk) instalado em sua máquina. Utilize o Makefile fornecido.
